Функции

Функции - огромная тема для любого языка программирования, здесь мы рассмотрим только самые основные моменты. Говоря простым языком, кусок кода, завернутый в специальную оболочку и имеющий имя, по которому можно к нему обращаться, называется функцией. Делается это для того, чтобы не прописывать одну и ту же последовательность действий каждый раз, а просто вызывать функцию.

Рассмотрим самую простую функцию, выполняющую сложение двух элементов.

>>> def sum(a,b): #задаем функцию с именем sum от двух переменных a и b
...     return(a+b) #возвращаем значение суммы a и b
... 
>>> sum(1,1) #теперь мы можем вызвать функцию, просто написав ее название
2 #и получим результат выполнения

Внимательно посмотрим на код:

>>> c = 4
>>> d = 5
>>> sum(c,d)
9
>>> a = 3
>>> b = 4
>>> sum(c,d)
7

Мы можем давать функции не только сами объекты, но и ссылки на них, то есть имена переменных.

Пришло время разобраться с тем, что такое глобольные и локальные переменные. Глобальные переменные используются в теле программы, локальные - существуют только внутри функции. Пример:

>>> def sum(a,b): 
...     print(a) #слегка изменим код функции, добавив вывод локальной переменной a
...     return(a+b)
...
>>> sum(b,a): 
4 #глобальная переменная b стала локальной переменной a
7

А что если у нас есть, например, константа k, и мы хотим использовать ее в нашей функции, а подавать на вход, например не очень хотим. В этом нам поможет global.

>>> k = 3
>>> def sum(a,b):
...     global k
...     return((a+b)*k)
... 
>>> sum(1,1)
6

Ну и конечно же функции являются основой такого явления, как рекурсия!

>>> def countdown(a):
...     print(a)
...     if a > 0:
...         countdown(a-1)
... 
>>> countdown(5)
5
4
3
2
1
0
>>> def countup(a):
...     if a > 0:
...         countup(a-1)
...     print(a)
... 
>>> countup(5)

4
5
